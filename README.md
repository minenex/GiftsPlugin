# GiftsPlugin

Плагин для Minecraft Paper 1.21.8+, позволяющий размещать и находить подарки с наградами.

## Возможности

- Создание обычных и золотых подарков с крутыми текстурами
- Размещение подарков в мире для поиска другими игроками
- Награды за нахождение подарков (предметы и эффекты)
- Праздничные фейерверки при нахождении подарка
- Удобное GUI-меню для просмотра всех подарков
- Телепортация к подаркам из меню
- Полностью настраиваемые награды и сообщения

## Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/gifts give` | Получить обычный подарок | `gifts.admin` |
| `/gifts supergift` | Получить золотой подарок | `gifts.admin` |
| `/gifts info` | Открыть меню подарков | `gifts.use` |
| `/gifts reload` | Перезагрузить конфигурацию | `gifts.admin` |

## Права

| Право | Описание | По умолчанию |
|-------|----------|--------------|
| `gifts.use` | Доступ к командам плагина | OP |
| `gifts.admin` | Управление подарками | OP |
| `gifts.teleport` | Телепортация к подаркам | OP |

## Установка

1. Скачайте JAR-файл плагина
2. Поместите его в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Настройте плагин в файле `config.yml`

## Конфигурация

Плагин создает следующие файлы:
- `config.yml` - основные настройки
- `gifts.yml` - данные о подарках (автоматически)

### Настройка наград

```yaml
rewards:
  normal:
    items:
      - material: DIAMOND
        amount: 1
    effects: []
  super:
    items:
      - material: DIAMOND
        amount: 64
    effects:
      - type: REGENERATION
        duration: 600
        amplifier: 1
```

### Настройка фейерверков

```yaml
firework:
  power: 1
  colors:
    - RED
    - YELLOW
    - ORANGE
  fade-colors:
    - WHITE
  type: BALL_LARGE
  flicker: true
  trail: true
```

## Использование

1. Получите подарок командой `/gifts give` или `/gifts supergift`
2. Кликните ПКМ по блоку, чтобы установить подарок
3. Другие игроки могут найти подарок, кликнув по нему ПКМ
4. При нахождении игрок получит награды и увидит фейерверки

## Требования

- Paper 1.21.8+
- Java 21+

## Сборка

```bash
./gradlew build
```

Собранный JAR-файл будет находиться в `build/libs/`.
